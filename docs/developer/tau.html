tau package¶

TAU Commander package initialization.

The first line of this file is the first thing to be executed whenever TAU Commander is invoked in any way. This module establishes global constants (as few as possible) and checks the Python version. It must be kept as short and simple as possible.

tau.EXIT_FAILURE = -100¶

int: Process exit code indicating unrecoverable failure.

tau.EXIT_SUCCESS = 0¶

int: Process exit code indicating successful operation.

tau.EXIT_WARNING = 100¶

int: Process exit code indicating non-optimal condition on exit.

tau.HELP_CONTACT = '<support@paratools.com>'¶

str: E-mail address users should contact for help.

tau.MINIMUM_PYTHON_VERSION = (2, 7)¶

tuple: Minimum required Python version for TAU Comamnder.

A tuple of at least (MAJOR, MINOR) directly comparible to sys.version_info

tau.PROJECT_DIR = '.tau'¶

str: Name of the project-level directory containing TAU Commander project files.

tau.SYSTEM_PREFIX = '/home/travis/build/zbeekman/taucmdr/.system'¶

str: System-level TAU Commander files.

tau.TAUCMDR_URL = 'http://www.taucommander.com/'¶

str: URL of the TAU Commander project.

tau.TAU_HOME = '/home/travis/build/zbeekman/taucmdr'¶

str: Absolute path to the top-level TAU Commander directory.

This directory contains at least bin, docs, and packages directories and is the root for system-level package installation paths. Do not change it once it is set.

tau.TAU_SCRIPT = 'sphinx'¶

str: Script that launched TAU Commander.

Mainly used for help messages. Do not change it once it is set.

tau.USER_PREFIX = '/home/travis/.tau'¶

str: User-level TAU Commander files.

tau.logger module¶

TAU Commander logging.

TAU Commander has two channels for communicating with the user:
  1. sys.stdout via The print statement. Use this for messages the user has requested, e.g. a project listing.
  2. sys.stdout and sys.stderr via tau.logger. Use this for status messages generated by TAU Commander.

TAU Commander also logs all status messages at the highest reporting level to a rotating debug file in the user’s TAU Commander project prefix, typically “~/.tau”.

Width of a line on the terminal.

tau.util.walk_packages(path, prefix)[source]¶

Fix pkgutil.walk_packages to work with Python zip files.

Python’s default zipimporter doesn’t provide an iter_modules method so pkgutil.walk_packages silently fails to list modules and packages when they are in a zip file. This implementation works around this.
